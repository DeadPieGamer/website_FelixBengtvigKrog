Premise

Web-game (wip) is a small tic-tac-toe game, made so that I could try out some different js stuff. The only sources of help while making this were the documentations of the different libraries and frameworks used.

Only move is allowed every five minutes. Every player shares the same board and moves. Hope you'll enjoy! :D (given this being a locally run site, and not accessible online, this effectively just means you have to wait 5 minutes between each move. The board is stored between sessions though)
